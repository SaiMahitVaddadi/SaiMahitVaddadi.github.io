  1. 01 2023–present active

    LillyPROP — Molecular Property Prediction Platform

    Platform engineer and research lead on a production ML system for ADME and toxicity prediction across small molecules, peptides, and antibodies.

    Read more

    LillyPROP is Eli Lilly's internal platform for predicting molecular properties at scale — covering ADME endpoints, toxicity flags, and off-target risks across three modality classes: small molecules, therapeutic peptides, and antibodies. The platform handles thousands of predictions per week across the discovery organization.

    The model layer combines graph neural networks (DGL/PyG) with pre-trained molecular and biological language models — ChemBERTa for small molecules, ESM-2 for proteins, ABLang-2 for antibody sequences — through multimodal fusion architectures that let evidence from different encoders reinforce each other.

    My current focus is the out-of-distribution detection subsystem: a gating layer that flags predictions where the input lies outside the model's reliable applicability domain. This is the difference between a model that is sometimes wrong and a model that knows when it is wrong.

  2. 02 2024–present active

    Out-of-Distribution Detection for Production ML

    A four-aim research program for reliable molecular property prediction — from structure–activity landscape methods to multimodal disagreement signals.

    Read more

    ML models in drug discovery are wrong constantly. The question is whether they know they're wrong. This research program addresses that directly through four aims: (1) structure–activity landscape index methods for quantifying prediction confidence; (2) assay-quality-aware uncertainty that propagates experimental noise through the model; (3) model-based Mahalanobis distance with cross-modal disagreement as an OOD signal; (4) microservice integration to gate production predictions.

    The first two aims are currently shipping into the LillyPROP platform. The full program is being written up for submission to ACS Central Science or JCIM.

  4. 04 2024 open source

    MolView — Desktop Molecular Identifier App

    A Tauri desktop application for resolving molecular identifiers, with an embedded MCP server for use from AI agents.

    Read more

    MolView resolves molecular identifiers across InChI, SMILES, IUPAC name, CAS number, and structure search — a daily friction point for chemists who work across databases with inconsistent identifier schemes.

    Built with Tauri for the shell, React for the UI, FastAPI for the Python backend, and RDKit for the chemistry core. An embedded MCP server exposes the resolution engine to AI agents, so a Claude session can look up and convert structures without leaving the conversation.

  5. 05 2023–present writing

    Technical Reference Library

    A 30+ document Feynman-style LaTeX series on ML topics written from a chemical-engineering-first perspective.

    Read more

    Most ML education is written for computer scientists. I write for people who came up through chemical engineering, process modeling, and physical chemistry — the audience that needs GNN message passing explained through the analogy of transport phenomena, or transformer attention through the lens of linear operators.

    The series now covers GNN architectures, transformer pretraining objectives, conformal prediction, MLIP integration, outlier detection in high-dimensional spaces, and aptamer generative model steering. Used internally for onboarding new computational scientists and as a reference for the LillyPROP team.
